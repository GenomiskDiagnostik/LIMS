# Assumptions

1. `GlimR.html` er den primære runtime og leveres som én fil.
2. Eksisterende README beskriver store dele af nuværende funktionalitet, men er ikke tilstrækkelig som kodebevis.
3. Videreudviklingen skal ske med stærkt fokus på stabilitet frem for hastighed.
4. Dansk er standard sprog.
5. Projektet skal kunne overtages af andre udviklere eller leverandører.
6. Kliniske workflows tåler dårligere flashy features end dårlige regressioner.
